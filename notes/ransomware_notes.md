# Ransomware Overview

## What it is
Malware that encrypts files and demands payment for decryption.

## Behavior
- Encrypts files (often ZIP, AES, or custom formats)
- Drops ransom note
- Demands cryptocurrency payment

## Recovery options
- Restore from backups
- Use decryptors if available
- Identify ransomware strain using tools

## Important tools
- ID Ransomware
- No More Ransom Project

## Key lesson
Prevention and backups are more effective than recovery.
